Python Weather API Assignment




Project Summary

The goal of this assignment was to analyze how weather trends change as latitude approaches the equator using live weather data.

The first step was to use a python library called Citipy to compile a list of randomly-selected cities, including the name of the city and it's latitude/longitude coordinates. While sample size tended to vary slightly, the sample size typically fell around 600 cities. This city data was then used to perform a series of API calls to the OpenWeatherData API to retrieve current weather statistics in real-time. The results of these API calls were then loaded into python for analysis using the Pandas library, as well as being saved to a .csv to preserve the raw data for later reference.

Because data couldn't be found for every city in the randomly-generated sample, data with null records was filtered-out. The refined data was also saved as a .csv file for later reference. Using this refined data, I used the MatPlotLib library to create scatter-plots that show weather trends in relation to latitude.

Each graph can be viewed individually by using the "Visualizations" menu to the right, while both datasets can be viewed using the "Data" menu above.
